# 轻量迭代任务清单: feat-restart-core-button

目标: 在主题页增加“手动重启内核”按钮，允许用户在不改动配置的情况下主动触发一次内核重启/启动，并在系统代理开启时自动做安全处理（避免指向黑洞断网）。

## 任务

- [√] 主题页（dark/light）：在首页增加“重启内核”按钮入口
- [√] 主题页（dark/light）：实现按钮逻辑（必要时关闭系统代理 → `POST /proxy/start` → 刷新状态/日志 → 尝试恢复系统代理）
- [√] Electron：内置主题同步策略升级（避免 userData 缓存导致 UI 变更不可见）
- [√] 更新知识库与变更索引（CHANGELOG / history / modules）

## 备注

- 复用既有 `POST /proxy/start` 行为：内核运行中调用将触发一次重启；未运行则等价于启动。
